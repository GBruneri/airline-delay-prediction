# Airline Delay Prediction

Projeto de Ciência de Dados com foco em previsão de atrasos de voos utilizando dados históricos agregados de companhias aéreas e aeroportos dos EUA.

## Objetivo
Construir um pipeline de dados e modelos de Machine Learning capazes de prever o atraso médio por voo em um determinado mês, considerando informações históricas, temporais e operacionais.

## Dataset
Airline Delay Cause (Kaggle)

## Estrutura do Projeto
- data/          : dados brutos e processados
- notebooks/     : exploração e análises (EDA)
- src/           : código de produção (ETL, treino, inferência)
- models/        : modelos treinados
- reports/       : métricas e visualizações
- tests/         : testes automatizados
